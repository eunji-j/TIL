### 4. 기초 수식 

(시험)★

-문제1. T(n) = T(n-1) + **1**

```
T(n) = T(n-1) + 1
	 = T(n-2) + 1 + 1
	 = T(1) + 1 +...+ 1
	 
T(n) = O(n)
```

-문제2. T(n) = T(n-1) + **n**

```
T(n) = T(n-1) + n
	 = T(n-2) + (n-1) + n
	 = T(1) +...+ (n-1) + n
	 = 1 +...+ (n-1) + n
	 = n(n+1)/2
	 
T(n) = O(n**2)
```

-문제3. T(n) = T(n-1) + **log n**

```
T(n) = T(1) + log n + log n-1 + log n-2 +...+ log 2
	 <= T(1) + log n + log n + log n +...+ log n
	 <= T(1) + nlog n
	 
T(n) = O(nlog n)
```

-문제4. T(n) = T(n/2) + 1

```
T(n) = T(n/2) + 1
	 = (T(n/4)+1) + 1
	 2**x = n
	 x = log n
	 
T(n) = O(log n)
```

-문제5. T(n) = T(n/2) + n

```
T(n) = T(n/2) + n
	 = (T(n/4)+n/2) + n
	 
T(n) = O(n)	
```
